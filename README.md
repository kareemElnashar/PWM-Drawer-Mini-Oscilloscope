# PWM Drawer (Mini Oscilloscope)

A PWM measurement and visualization system implemented using ATmega32.

## Features
- PWM Frequency Measurement
- Duty Cycle Measurement
- Ton and Toff Calculation
- Time Period Measurement
- Internal / External Source Selection
- LCD Visualization
- Timer1 Input Capture Technique

## Hardware
- ATmega32
- LCD 20x4
- Proteus Simulation
- Timer1 Input Capture

## Software
- Embedded C
- AVR-GCC
- Proteus
